Swords and Spells is a 3D action RPG developed in Unity, featuring both singleplayer and online cooperative multiplayer modes. Set in a medieval fantasy world, players can choose between different character classes, each with unique abilities and playstyles. The main objective is to progress through the level by defeating enemies and bosses, unlocking new abilities at checkpoints, and experiencing dynamic and engaging cooperative gameplay.

This game was developed as the final project of a Bachelor’s degree by Ricardo Vieira and Gonçalo Alves.

The scripts presented are organized into two distinct folders:

Multiplayer
- Contains all scripts used in the Multiplayer context, including character classes, enemies, Multiplayer implementation logic, spawners, and UI elements present in the game, associated with the folder with the corresponding name.

Single Player

- Contains all scripts used in the Single Player context, following a structure similar to the "Multiplayer" folder.
